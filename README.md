# Project Title
Team Projects Management Tool

## Team Members
Seth Taylor, Kaden Boam, Matthew Morton, Jacob Strong

## Software Description
Solution for team projects, with accountability tracking per sprint or unit via points and pie charts.

## Architecture
Programming language: Java Script, React
Framework: Web Application, CSS
Data Storage: Supabase
Development Tools: Github, VScode, Github Pages

## Communication
Discord group message and in person.

## Responsibilities
Conducting Meetings: Seth
Maintaining Team Assignment List: Seth
Ensuring Github is working: Kaden
Maintaining Documentation: Seth Taylor
Create & Display Presentations: Matthew
Submit Team Assignments: Jacob / Anyone

## Running Locally
Setting up supabase connection by running this command:  npm install @supabase/supabase-js
To run the website locally, use this command: npm run dev

## Reflection
What the team Learned?
The complications of building web applications became very apparent. A litte change could cause the page to refuse to run outright. We learned how to better us React. We also learned how to use Supabase. We also touched upon node.js, to store as well as set up the website values.

What can be Improved?
Out assigmnet warning is hardcoded, and doesn't change depending on sprints or projects. Our pie charts are new and could use some formating. Breaking the page could be very easy to do if you know your way behind the code. We wanted a working product, but have not put much thouhgt into data use and loading time, so some adjustments for those would be good.

Future Plans for this Project?
We were not able to accomplish everything that we originally had planned. Push notifications are close with the warning system we have in place, but still needs to be implemented. User logins needs to be added on to the database. Making it into an extension would also be nice. Being able to import from other websites like canvas would be worth future investment as well.
